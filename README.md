# Neon Stack 3D

Created by iben.

Game bergaya Tetris 3D untuk Godot 4. Jalankan `project.godot`, lalu tekan Play.

## Kontrol

- `A` / `Left`: geser kiri
- `D` / `Right`: geser kanan
- `S` / `Down`: soft drop
- `W` / `Up`: rotasi
- `Space`: hard drop
- `Q` / `E`: putar kamera
- Drag mouse kanan: orbit kamera
- Scroll mouse: zoom kamera
- `C`: ganti preset kamera
- `P` / `Esc`: pause menu
- `R`: restart

Game memakai musik `Voltaic` by Kevin MacLeod sebagai track utama, dengan fallback EDM procedural loop jika file audio tidak bisa diload.

## Level

Level naik otomatis saat skor melewati target berikutnya. Setiap level membuat blok jatuh lebih cepat dan mengganti background parallax.
